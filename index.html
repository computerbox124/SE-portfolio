About Me

My Journey

My journey in programming started in 5th grade when I built my first calculator application using Visual Basic. That simple project sparked a passion that led me to game development, competitive programming, and eventually to software engineering.

In 2015, I received the WIPO Gold Medal from the World Intellectual Property Organization for developing an educational application that helps teachers calculate student assessments efficiently. This award recognized my work in leveraging IT and programming to make education more productive and engaging.

Also in 2016, I created my own first-person shooter game "I.N.T.E.R.E.T.U.R.N of Karabakh" using the X-Ray Engine — the same legendary engine powering the famous S.T.A.L.K.E.R. game series. I designed custom game mechanics, levels, and even produced my own teaser trailers.

My competitive programming background has trained me to solve, debug, and fix problems efficiently, even with minimal prior knowledge of specific frameworks. I learn quickly and find the information I need to get things done. It was a great experience building my first apps and games without AI or external help, which taught me self-reliance and deep problem-solving skills.

Today, I build high-performance backend systems, craft platforms for education, and develop tools that empower developers to work more efficiently. I leverage AI in my work to make development more efficient and productive, while bringing the same determination and adaptability that helped me succeed in competitive programming. I've had the privilege of representing Azerbaijan at IOI 2019, reaching ICPC Semifinals, and building platforms that have been used by thousands of programmers in major competitions like IDDA Cup, Azercell Cup, and ARTI.

Online Exam Proctoring System

2020 - 2021 (frozen), 2026 - Present

Examinator

A proctor for organizing exams, originally created as an alternative to Lock-down browser. Unlike traditional proctors where students lose exam progress if internet disconnects, Examinator was designed to handle such scenarios gracefully. Initially built with Python and tkinter in 2020, the project was frozen when universities showed no interest in alternatives. Now restored for CodeAny to enable fair participation control in online competitions, using Rust + Tauri + Vue3 for the client. Currently in development.

  • •Online exam proctoring without internet dependency issues
  • •Alternative to traditional lock-down browsers
  • •Fair participation control for competitions
